##  Запуск проекта в режиме разработки

Для работы приложения необходимо открыть **три отдельных окна терминала** (или три вкладки в VS Code), так как сервер, клиент и разовый скрипт должны работать независимо.

### Шаг 1: Установка зависимостей и запуск Сервера (Backend)
В первом окне терминала перейдите в папку бэкенда, установите пакеты и запустите сервер. При первом запуске ORM автоматически создаст необходимые таблицы в вашей базе данных PostgreSQL.

```bash
cd backend
npm install
npm run dev
