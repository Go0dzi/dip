## 🚀 Инструкция по запуску проекта

Для успешного запуска приложения вам понадобятся установленные **Node.js** и **PostgreSQL**.

### 1. Настройка базы данных и переменных окружения
Перед запуском убедитесь, что вы создали файл `.env` в папке `backend` и указали там данные для подключения к вашей локальной базе данных PostgreSQL.

### 2. Запуск серверной части (Backend)
Откройте терминал, перейдите в директорию сервера, установите зависимости (если запускаете впервые) и запустите сервер в режиме разработки:

```bash
cd backend
npm install
cd backend
npm run dev
cd frontend
npm run dev
cd backend
node createAdmin.js
